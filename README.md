## Stats

Your dog has three main stats that you need to manage:

- **Hunger** - Decreases over time, feed your dog to restore it
- **Happiness** - Decreases over time, play and pet your dog to keep them happy
- **Energy** - Decreases over time (and when playing), let your dog sleep to restore it

## How to Play

1. Open `index.html` in your web browser
2. Take care of your virtual dog by clicking the action buttons
3. Monitor the stat bars and keep them from getting too low
4. Your progress is automatically saved in your browser

## Installation

No installation required! Simply:

1. Clone or download this repository
2. Open `index.html` in any modern web browser
3. Start playing!

```bash
# Clone the repository
git clone https://github.com/YannickLongval/dog.git

# Open the game
cd dog
open index.html  # or double-click the file
```
